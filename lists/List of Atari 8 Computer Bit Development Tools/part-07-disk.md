## a8rawconv

**Official URL**
https://github.com/Axel051171/a8rawconv-0.95

**Host Platforms**

- Windows
- Linux
- macOS

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
a8rawconv is a command-line utility for converting between Atari 8-bit disk image formats and raw disk data. It is primarily intended for development, preservation, and hardware projects where direct access to disk sectors or low-level media layouts is required. The utility integrates well into automated build pipelines and image preparation workflows.

**Development Features**

- Disk image conversion
- Raw disk extraction
- Command-line interface
- Batch processing
- Cross-platform support

**Supported File Formats**

- .ATR
- Raw disk images

**Integration**

- Build pipelines
- Disk image utilities

**Primary Purpose**  
Command-line utility for converting Atari 8-bit disk images and raw media.

## RespeQt

**Official URL**
https://github.com/RespeQt/RespeQt

**Host Platforms**

- Windows
- Linux
- macOS

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
RespeQt is a modern cross-platform disk and virtual peripheral utility for Atari 8-bit computers. It enables developers to create, inspect, modify, and transfer Atari disk images while emulating Atari serial peripherals using the SIO protocol. The application supports a wide range of development and testing workflows and is commonly used with both emulators and real hardware.

**Development Features**

- ATR disk image management
- File browser
- Disk formatting
- Virtual SIO devices
- Printer emulation
- Serial communication
- Cross-platform interface

**Supported File Formats**

- .ATR
- .ATX
- .CAS
- .XEX

**Integration**

- Emulators
  - Altirra
  - Atari800
- Real Atari hardware
- SIO interfaces

**Primary Purpose**  
Cross-platform Atari disk image manager and virtual peripheral utility.
