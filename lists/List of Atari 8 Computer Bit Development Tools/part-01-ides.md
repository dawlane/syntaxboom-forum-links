## ATasm Altirra Bridge

**Official URL**
https://marketplace.visualstudio.com/items?itemName=cerebus.atasm-altirra-bridge

**Host IDE**

* Visual Studio Code

**Host Platforms**

* Windows
* Linux
* macOS

**Verified Target Systems**

* Atari 8-bit Family

**Overview**
ATasm Altirra Bridge is a Visual Studio Code extension that transforms Visual Studio Code into a complete Atari 8-bit cross-development environment. It integrates the ATasm assembler with the Altirra emulator, allowing source code to be assembled, executed, debugged, and iterated without leaving the editor.

The extension provides an integrated workflow for assembly language development, including configurable build tasks, emulator launching, symbol handling, and source-level debugging through Altirra. It is intended for developers targeting Atari 8-bit computers using modern development practices while retaining compatibility with established Atari development tools.

As a Visual Studio Code extension, it benefits from the editor's project management, source control integration, extension ecosystem, and cross-platform support while adding Atari-specific development functionality.

**Development Features**

* Integrated ATasm assembler support
* Optional MADS assembler support
* Build task integration
* Altirra emulator launching
* Source-level debugging
* Breakpoint support
* Symbol navigation
* Error reporting
* Syntax highlighting
* Project-based development
* Visual Studio Code task integration

**Supported File Formats**

* .ASM
* .LST
* .XEX

**Integration**

* Assemblers

  * ATasm
  * MADS
* Emulators

  * Altirra
* External Tools

  * Visual Studio Code

**Primary Purpose**
Visual Studio Code extension providing an integrated Atari 8-bit assembly development environment with build, emulator, and debugging support.

## WUDSN IDE

**Official URL**
https://www.wudsn.com/index.php/ide

**Source Code**
https://github.com/peterdell/wudsn-ide

**Host Platforms**

* Windows
* Linux
* macOS

**Verified Target Systems**

* Atari 8-bit Family

**Overview**
WUDSN IDE (Workbench for User-friendly Development of Small Native Systems) is an Eclipse-based Integrated Development Environment designed for cross-development of software for classic 8-bit computer systems, including the Atari 8-bit family. Rather than replacing existing assemblers and compilers, it integrates them into a unified workflow that simplifies project management, editing, building, and emulator execution.

For Atari development, WUDSN IDE supports popular development tools such as MADS, ATASM, XASM, and Mad Pascal. It provides syntax-aware editing, project management, configurable build pipelines, compiler and assembler integration, source navigation, error parsing, and emulator launching. The IDE is designed to reduce development setup time while providing a consistent workflow across supported host platforms.

WUDSN IDE also includes project templates, configuration management, and integration with multiple emulators, making it one of the most comprehensive dedicated development environments available for Atari 8-bit cross-development.

**Development Features**

* Eclipse-based integrated development environment
* Project management
* Syntax highlighting
* Source outline view
* Source folding
* Hyperlink navigation
* Configurable build system
* Automatic assembler and compiler error parsing
* Integrated emulator launching
* Graphics conversion editor
* Hex editor
* Build annotations
* Example projects
* Cross-platform installation

**Supported File Formats**

* .ASM
* .ASX
* .PAS
* .XEX

**Integration**

* Assemblers

  * ATASM
  * MADS
  * XASM
* Cross-Compilers & High-Level Languages

  * Mad Pascal
* Emulators

  * Altirra
  * Atari800
  * Atari800MacX
* External Tools

  * Eclipse

**Primary Purpose**
Standalone Integrated Development Environment for Atari 8-bit cross-development, providing a unified workflow around assemblers, compilers, and emulators.
