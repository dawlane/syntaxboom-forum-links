## DefleMask

**Official URL**
https://www.deflemask.com/

**Host Platforms**

- Windows
- Linux
- macOS

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
DefleMask is a cross-platform tracker designed for composing music for a wide range of classic computer and console sound chips. Through its Atari POKEY support, it enables musicians and developers to create music and sound effects suitable for Atari 8-bit software development. DefleMask offers a modern tracker interface, real-time playback, instrument editing, and export facilities that integrate well into contemporary cross-development workflows.

**Development Features**

- Multi-system tracker
- POKEY sound chip support
- Instrument editor
- Pattern editor
- Order editor
- Effect commands
- Real-time playback
- Sample import (where supported)
- Cross-platform user interface

**Supported Playback Engines**

- POKEY

**Supported File Formats**

- .DMF
- .WAV

**Integration**

- Asset Conversion Tools
- Modern audio workflows

**Primary Purpose**  
Cross-platform tracker supporting Atari POKEY music composition.

## Raster Music Tracker

**Official URL**
https://github.com/raster-atari-org/RASTER-Music-Tracker

**Source Code**
https://github.com/raster-atari-org/RASTER-Music-Tracker

**Host Platforms**

- Windows

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
Raster Music Tracker (RMT) is the de facto standard music tracker for Atari 8-bit development. Designed specifically for the Atari POKEY sound chip, it provides an extensive music composition environment together with playback routines that are widely used in commercial-quality games, demos, and homebrew software. The project now appears to be maintained at the raster-atari-org/RASTER-Music-Tracker repository, and it remains one of the most important tools in the Atari development ecosystem.

**Development Features**

- Native POKEY tracker
- Instrument editor
- Pattern editor
- Song editor
- Effect commands
- Real-time playback
- Playback routine generation
- Music testing within the editor
- Export for integration into software projects

**Supported Playback Engines**

- Raster Music Tracker Player

**Supported File Formats**

- .RMT
- .SAP

**Integration**

- Assemblers
  - MADS
  - ATasm
- Cross-Compilers & High-Level Languages
  - cc65
  - FastBasic
  - Mad Pascal

**Primary Purpose**  
Native Atari POKEY music tracker and playback engine for Atari 8-bit software development.
