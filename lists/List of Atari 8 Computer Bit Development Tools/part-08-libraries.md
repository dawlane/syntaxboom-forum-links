## blibs

**Official URL**
https://gitlab.com/bocianu/blibs

**Host Platforms**

- Windows
- Linux
- macOS

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
blibs is an independently distributed collection of reusable Mad Pascal libraries for Atari 8-bit software development. It extends the standard Mad Pascal runtime with a broad range of modules covering graphics, Player/Missile Graphics, display lists, operating system services, SIO communication, xBIOS support, networking, compression, utility functions, and other common development tasks. The libraries are designed to be reused across multiple projects and are supplied with comprehensive API documentation and example code.

Unlike the libraries supplied with the Mad Pascal compiler itself, blibs is maintained as an independent project with its own documentation and release history, making it suitable for inclusion as a standalone framework within the Atari development ecosystem.

**Provided Libraries / Modules**

- Graphics utilities
- Player/Missile Graphics (PMG)
- Display List management
- CRT utilities
- SIO interface
- xBIOS interface
- Networking (FujiNet support)
- Utility functions
- Compression helpers
- File and system utilities

**Development Features**

- Reusable library collection
- Comprehensive API documentation
- Example programs
- Modular design
- Cross-platform development
- Designed for Mad Pascal

**Supported File Formats**

- .PAS
- .PP

**Integration**

- Cross-Compilers & High-Level Languages
  - Mad Pascal

**Primary Purpose**  
Standalone collection of reusable libraries for Mad Pascal Atari 8-bit software development.
