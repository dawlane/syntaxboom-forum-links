## Exomizer

**Official URL**
https://bitbucket.org/magli143/exomizer/src/master/

**Host Platforms**

* Windows
* Linux
* macOS

**Verified Target Systems**

* Atari 8-bit Family

**Overview**
Exomizer is a widely used cross-platform compression and executable crunching utility for 6502-based systems. It supports efficient compression of binaries and data files and includes self-extracting executable generation. Although developed for multiple 6502 platforms, it is commonly used in Atari 8-bit development because of its excellent compression ratios and mature toolchain support.

**Development Features**

* Executable crunching
* Data compression
* Self-extracting executables
* Command-line interface
* Cross-platform support
* Batch processing

**Supported File Formats**

* Input

  * Binary files

* Output

  * Exomizer compressed data
  * Self-extracting executables

**Integration**

* Assemblers

  * MADS
  * ATasm
* Cross-Compilers & High-Level Languages

  * cc65
  * Mad Pascal
* Build systems

**Primary Purpose**
Cross-platform executable cruncher and data compression utility for 6502 software development.
