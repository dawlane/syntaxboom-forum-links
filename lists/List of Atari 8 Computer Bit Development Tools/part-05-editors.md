## Atari MapMaker

**Official URL**
https://sourceforge.net/projects/atari-mapmaker/

**Source Code**
https://sourceforge.net/p/atari-mapmaker/code/HEAD/tree/

**Host Platforms**

- Windows

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
Atari MapMaker is a dedicated map and level editor for Atari 8-bit software development. It is designed to simplify the creation of tile-based maps for games by providing an intuitive editing interface, tile palette management, and export facilities suitable for integration into assembly language and high-level language projects. Although capable of importing and exporting map data, its primary purpose is interactive map creation, making it best classified as a graphics editor rather than an asset conversion utility.

**Development Features**

- Tile map editor
- Tile palette management
- Multi-map projects
- Layered editing
- Import and export facilities
- Map preview
- Grid editing

**Supported File Formats**

- Atari MapMaker project files
- Exported map data

**Integration**

- Assemblers
  - MADS
  - ATasm
- Cross-Compilers & High-Level Languages
  - cc65
  - Mad Pascal

**Primary Purpose**  
Tile map editor for creating Atari 8-bit game levels and map data.

## Graph2Font

**Official URL**
https://g2f.atari8.info/

**Host Platforms**

- Windows
- Linux
- macOS

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
Graph2Font is one of the most widely used graphics development tools for the Atari 8-bit family. It combines a bitmap graphics editor, font editor, character editor, display list editor, and screen designer into a single integrated application. Designed specifically around Atari display hardware, it enables developers to create complex graphics utilising multiple graphics modes, Display List Interrupts (DLIs), custom character sets, and colour changes. Graph2Font has become a standard asset creation tool for modern Atari game, demo, and utility development.

**Development Features**

- Bitmap screen editor
- Character editor
- Font editor
- Display List editor
- DLI colour editing
- Multi-mode graphics support
- Screen preview
- Assembly data export
- Native Atari palette support
- Import and export facilities

**Supported File Formats**

- .FNT
- .G2F
- .PNG
- .XEX

**Integration**

- Assemblers
  - MADS
  - ATasm
- Cross-Compilers & High-Level Languages
  - Mad Pascal
  - cc65

**Primary Purpose**  
Integrated graphics, character, font, and screen editor for Atari 8-bit software development.

## RECOIL

**Official URL**
https://recoil.sourceforge.net/

**Source Code**
https://sourceforge.net/p/recoil/code/

**Host Platforms**

- Windows
- Linux
- macOS
- Web

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
RECOIL (Retro Computer Image Library) is a comprehensive image decoding library and viewer supporting hundreds of native graphics formats from classic computer systems, including the Atari 8-bit family. It enables developers to inspect, validate, preview, and verify Atari graphics assets throughout the development process. While not an editor itself, RECOIL is an invaluable companion tool for graphics development due to its extensive format support and accurate rendering of native Atari image formats.

**Development Features**

- Native Atari image decoding
- Graphics viewer
- Image conversion
- Batch processing
- Command-line tools
- Cross-platform library
- Web viewer
- Plugin support

**Supported File Formats**

- Numerous Atari-native bitmap, font and graphics formats
- .G2F
- .MIC
- .PIC

**Integration**

- Asset Conversion Tools
- Modern graphics editors
- Build pipelines

**Primary Purpose**  
Viewer and image decoding library for Atari-native graphics formats.

## SprEd

**Official URL**
https://bocianu.gitlab.io/spred/

**Source Code**
https://gitlab.com/bocianu/spred

**Host Platforms**

- Web

**Verified Target Systems**

- Atari 8-bit Family

**Overview**  
SprEd is a browser-based Player/Missile Graphics editor developed specifically for Atari 8-bit software development. It provides an intuitive interface for designing hardware sprites, previewing animation frames, and exporting graphics data suitable for assembly language and high-level language projects. As a web application, it requires no installation and can be used from any modern browser.

**Development Features**

- Player/Missile Graphics editor
- Animation preview
- Pixel editing
- Frame editing
- Undo and redo
- Data export
- Browser-based interface

**Supported File Formats**

- Native SprEd project format
- Assembly source export

**Integration**

- Assemblers
  - MADS
  - ATasm
- Cross-Compilers & High-Level Languages
  - cc65
  - Mad Pascal

**Primary Purpose**  
Web-based Player/Missile Graphics editor for Atari 8-bit software development.

**Source Code**
https://github.com/matosimi/SwSprEd

**Overview**  
SwSprEd (Software Sprite Editor) is a cross-platform editor for creating software sprites for Atari 8-bit computers. It focuses on producing sprite graphics intended for software-rendered animation rather than hardware Player/Missile Graphics. The editor provides pixel editing, animation preview, palette management, and export facilities suitable for modern Atari game development.

**Development Features**

- Software sprite editor
- Animation preview
- Pixel editing
- Colour palette management
- Frame management
- Import and export

- Sprite data export

**Integration**

  - MADS
  - ATasm
- Cross-Compilers & High-Level Languages
  - cc65
  - Mad Pascal

**Primary Purpose**  
Cross-platform software sprite editor for Atari 8-bit graphics development.
