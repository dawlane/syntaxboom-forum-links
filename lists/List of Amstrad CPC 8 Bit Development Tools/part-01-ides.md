**Overview**  
KC IDE is a Visual Studio Code extension that provides a complete retro-computing development environment centred around Z80-based systems, including the Amstrad CPC family. It combines source editing, project management, build automation, debugging support, emulator integration, and asset handling into a unified workflow within Visual Studio Code.

The extension is designed to simplify modern cross-development by allowing programmers to write and manage assembly language projects on contemporary operating systems while targeting original Amstrad CPC hardware and emulators. KC IDE supports project-oriented development, making it suitable for larger software projects such as games, demos, utilities, operating system components, and hardware experiments.

Rather than acting as a standalone editor, KC IDE extends Visual Studio Code with Amstrad-focused tooling and integrates with common assemblers, emulators, and build processes. This allows developers to use modern editing features such as syntax highlighting, source navigation, code completion, project search, version control integration, and task automation while retaining compatibility with traditional Amstrad CPC development workflows.

KC IDE is particularly useful for developers seeking an all-in-one environment that reduces the complexity of managing separate editors, command-line tools, build scripts, and emulator configurations.
