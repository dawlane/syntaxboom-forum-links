**Overview**  
GrafX2 is a modern pixel-art editor inspired by the classic Deluxe Paint workflow and is widely used throughout the retro-computing community. It provides a comprehensive environment for creating bitmap graphics, sprites, character sets, user-interface elements, tilesets, maps, loading screens, and other graphical assets commonly used in Amstrad CPC software.

The editor is particularly well suited to Amstrad CPC development because it supports indexed-colour workflows and allows artists to work within the constraints imposed by classic hardware. Developers can create assets intended for CPC display modes while taking advantage of modern editing tools, layer-like page management, animation support, palette control, and advanced drawing functions.

GrafX2 is frequently used in conjunction with asset conversion utilities that transform artwork into CPC-compatible binary formats. This separation of creation and conversion workflows aligns well with modern cross-development practices, allowing artists and programmers to collaborate efficiently.

Because it supports a wide range of image formats and operating systems, GrafX2 has become one of the most commonly used graphics tools in retro game development, demoscene productions, and software preservation projects.
