# CPCRSLib

**Official URL**
https://sourceforge.net/projects/cpcrslib/

**Host Platforms**

* Windows
* Linux
* macOS

**Verified Amstrad CPC Targets**

* Amstrad CPC 464
* Amstrad CPC 664
* Amstrad CPC 6128

**Overview**  
CPCRSLib is a reusable software development library for the Amstrad CPC platform. It provides programmers with a collection of routines, helper functions, and development infrastructure intended to accelerate creation of games and other software projects.

The framework abstracts many commonly required programming tasks, reducing the amount of low-level code that developers must create from scratch. By providing reusable components, CPCRSLib allows projects to be developed more rapidly while improving code consistency and maintainability.

The library is primarily aimed at assembly-language development and is often used by programmers who wish to focus on game logic and application functionality rather than repeatedly implementing common platform services.

Within a modern development workflow, CPCRSLib serves as a reusable foundation layer that can be integrated into custom projects and automated build systems.

**Development Features**

* Reusable software library
* Game development support
* Utility routines
* Hardware abstraction helpers
* Memory management helpers
* Graphics support routines
* Input handling routines
* Sound support facilities
* Project code reuse
* Assembly language integration
* Modular framework design
* Development acceleration

**Supported File Formats**

* ASM
* INC
* BIN
* Library source files

**Integration**

* RASM
* PASMO
* SjASMPlus
* WinAPE
* Arnold
* ACE-DL
* Retro Virtual Machine

**External Tools**

* KC IDE
* Build automation systems
* Asset conversion tools
* Source control systems

**Primary Purpose**  
Provide reusable development libraries and infrastructure for Amstrad CPC software projects.

**Verified Amstrad CPC Targets**

* Amstrad CPC 6128

**Overview**  
CPCTelera is one of the most comprehensive and widely adopted software development frameworks available for the Amstrad CPC. It is designed primarily for game development and provides an extensive collection of libraries, tools, examples, documentation, and build infrastructure.

The framework combines a development SDK with a complete cross-development toolchain. It includes support for C and assembly language programming, graphics handling, sound playback, memory management, input processing, interrupt management, firmware access, and many other hardware-related tasks.

CPCTelera significantly reduces the complexity of CPC development by providing tested and reusable components that encapsulate common programming requirements. Developers can focus on gameplay, content creation, and application logic rather than repeatedly implementing low-level hardware routines.

The framework is extensively documented and is used in hobby, educational, commercial homebrew, and demoscene projects. It is often considered the primary modern SDK for Amstrad CPC software development.

**Overview**  
CPCTools is a collection of development utilities, libraries, and supporting infrastructure designed to simplify modern Amstrad CPC software development. It forms part of the broader CPCSDK ecosystem and provides developers with tools that assist in asset preparation, media creation, project automation, and deployment workflows.

Rather than acting as a single-purpose library, CPCTools serves as a toolkit that supports multiple stages of the development lifecycle. It is commonly used in conjunction with assemblers, compilers, emulators, and build systems to automate repetitive tasks and streamline project management.

The toolkit encourages reproducible builds and modern software engineering practices by allowing developers to incorporate CPC-specific tooling into automated workflows and continuous integration environments.

For larger projects, CPCTools can significantly reduce manual effort associated with media preparation, packaging, testing, and release generation.
