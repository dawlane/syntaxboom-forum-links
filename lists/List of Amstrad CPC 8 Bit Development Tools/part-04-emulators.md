* Linux

**Verified Amstrad CPC Targets**

* Amstrad CPC 464
* Amstrad CPC 664
* Amstrad CPC 6128
* CPC Plus range
* GX4000

**Overview**  
ACE-DL is a modern Amstrad CPC emulator focused on accuracy, debugging capabilities, and support for both software development and preservation activities. It is designed to emulate the original CPC hardware closely while providing modern tools useful to programmers and advanced users.

The emulator supports the complete CPC family, including the Plus range and GX4000 console. It offers a variety of debugging and inspection facilities that make it suitable for software development, reverse engineering, hardware research, and testing.

For developers, ACE-DL provides a convenient environment for quickly testing software builds without requiring original hardware. Its development-oriented features allow detailed inspection of memory, processor state, hardware registers, and execution flow.

The emulator is particularly useful for projects targeting enhanced CPC hardware because of its support for the CPC Plus range and GX4000 systems.
