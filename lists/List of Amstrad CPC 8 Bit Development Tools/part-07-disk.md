# CPCDiskXP

**Official URL**
https://www.cpcmania.com/cpcdiskxp/cpcdiskxp.htm

**Host Platforms**

* Windows

**Verified Amstrad CPC Targets**

* Amstrad CPC 464
* Amstrad CPC 664
* Amstrad CPC 6128
* CPC Plus range

**Overview**  
CPCDiskXP is a specialised disk management utility designed for working with Amstrad CPC floppy disk formats and disk image files. It provides developers, preservationists, and hobbyists with tools for creating, modifying, examining, and transferring software between modern computers and CPC-compatible media.

The software supports management of CPC disk images and physical floppy disks through compatible hardware. It is commonly used during software development to prepare test media, distribute software releases, inspect file structures, and verify disk contents.

For modern cross-development workflows, CPCDiskXP acts as a bridge between development systems and original Amstrad hardware. Developers can create and maintain disk images on a contemporary PC and then deploy them to emulators or real systems.

The utility is also widely used in software preservation projects where examination and recovery of CPC disk contents are required.

**Development Features**

* Disk image creation
* Disk image editing
* File extraction
* File insertion
* Directory management
* Disk formatting
* Physical floppy disk support
* CPC filesystem management
* Batch file operations
* Disk content inspection
* Media preparation
* Software deployment workflow
* Preservation support

**Supported File Formats**

* DSK
* Extended DSK
* CPC disk formats
* Binary files
* CPC file formats

**Integration**

* PASMO
* RASM
* SjASMPlus
* WinAPE
* Arnold
* ACE-DL
* Retro Virtual Machine

**External Tools**

* KC IDE
* CPCTelera
* z88dk
* Build automation systems
* File transfer workflows

**Primary Purpose** 
Create, manage, inspect, and deploy Amstrad CPC disk images and floppy disk media.

# iDSK

**Official URL**
https://github.com/cpcsdk/idsk

**Host Platforms**

* Windows
* Linux
* macOS

**Verified Amstrad CPC Targets**

* Amstrad CPC 464
* Amstrad CPC 664
* Amstrad CPC 6128
* CPC Plus range
* GX4000 development workflows using associated media assets

**Overview**  
iDSK is a command-line disk image management utility developed as part of the CPCSDK ecosystem. It is designed to automate creation and manipulation of Amstrad CPC disk images and is particularly useful within modern build pipelines and continuous integration environments.

Unlike graphical disk utilities, iDSK focuses on automation and scripting. Developers can incorporate it directly into build processes to create disk images, add files, organise directory structures, and prepare release media without manual intervention.

This makes iDSK especially valuable for larger projects where repeatable builds and automated deployment processes are important. It is frequently used alongside assemblers, compilers, asset conversion tools, and framework-based development environments.

The utility plays a key role in modern CPC software production by enabling fully automated generation of distributable disk images from source code and assets.

* File insertion

* Build pipeline integration
* Scriptable workflows

* Continuous integration support

* Reproducible build support
