## FASE

**Official URL**
https://github.com/DSkywalk/fase

**Host Platforms**

- Windows
- Linux
- macOS

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum Next

**Overview**  
FASE (Framework for Adventure and Strategy Engine development) is a reusable ZX Spectrum Next software framework intended to simplify the development of larger projects through a collection of common systems, libraries and development infrastructure. Rather than providing a complete game creation environment, FASE supplies reusable building blocks that developers can integrate into their own software projects.

The framework is designed to reduce development time by providing pre-built functionality and a structured project architecture. This allows developers to focus on game logic and content creation while leveraging existing infrastructure for common tasks. FASE is particularly useful for medium to large ZX Spectrum Next projects that benefit from a modular and maintainable codebase.

**Overview**

- ZX Spectrum 48K
- ZX Spectrum 128K
- ZX Spectrum +2
- ZX Spectrum +2A/+2B
- ZX Spectrum +3

**Overview**  
NIRVANA ENGINE is a reusable graphics engine for the Sinclair ZX Spectrum that enables multicolour sprite-style graphics beyond the limitations of standard attribute-based rendering. It provides a runtime system that developers can integrate into games requiring colourful animated objects on classic Spectrum hardware.

The engine abstracts much of the complexity involved in advanced Spectrum graphics programming and has been used in numerous homebrew projects. It is particularly valuable for developers seeking enhanced visual presentation while maintaining compatibility with original Spectrum machines.
