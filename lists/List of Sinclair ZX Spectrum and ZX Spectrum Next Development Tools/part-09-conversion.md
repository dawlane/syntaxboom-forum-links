## Gfx2Next

**Official URL**
https://www.baker76.com/gfx2next/

**Host Platforms**

- Windows

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum Next

**Overview**  
Gfx2Next is a specialised asset conversion utility designed for ZX Spectrum Next graphics development. It converts artwork created in modern graphics packages into formats directly usable by ZX Spectrum Next hardware and software projects. The tool is particularly popular among developers who create artwork in external editors such as GrafX2 and then require automated conversion into Next-compatible graphics resources.

The utility simplifies the process of preparing sprites, tiles, palettes and screen graphics for deployment on ZX Spectrum Next hardware. By automating conversion tasks and reducing manual processing, Gfx2Next helps maintain efficient asset pipelines and encourages iterative graphics development workflows.

**Development Features**

- Graphics conversion
- Sprite conversion
- Tile conversion
- Palette conversion
- Batch processing support
- ZX Spectrum Next format generation
- Development pipeline integration
- Asset preparation workflows

**Conversion Targets**

- ZX Spectrum Next sprites
- ZX Spectrum Next tiles
- ZX Spectrum Next palettes
- ZX Spectrum Next screen graphics

**Supported File Formats**

- `.png`
- ZX Spectrum Next graphics formats
- Palette resource formats

**Integration**

- **Assemblers**
  - SjASMPlus
- **Compilers**
  - z88dk
  - Boriel BASIC
- **Frameworks & SDKs**
  - Compatible with ZX Spectrum Next graphics frameworks
- **External tools**
  - GrafX2
  - ZX PearPixel

**Primary Purpose**  
A graphics asset conversion utility for transforming modern artwork into ZX Spectrum Next-compatible graphics resources.

## Remy's ZX Spectrum Tools

**Official URL**
https://zx.remysharp.com/tools/

**Host Platforms**

- Web

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 48K
- ZX Spectrum 128K
- ZX Spectrum +2
- ZX Spectrum +2A/+2B
- ZX Spectrum +3
- ZX Spectrum Next

**Overview**  
Remy's ZX Spectrum Tools is a collection of browser-based asset conversion and development utilities designed to assist ZX Spectrum and ZX Spectrum Next software creation. The suite provides a variety of conversion tools that transform modern assets into formats suitable for use on Sinclair hardware, reducing the need for custom scripts or manual conversion processes.

Because the tools are web-based, they can be accessed from any modern operating system without installation. The collection complements other browser-based tools developed by Remy Sharp and forms part of a lightweight, accessible development workflow suitable for rapid experimentation, prototyping and production use.

**Development Features**

- Browser-based operation
- Asset conversion utilities
- Graphics conversion
- Data conversion
- ZX Spectrum format generation
- ZX Spectrum Next format generation
- Rapid prototyping support
- Cross-platform accessibility

**Conversion Targets**

- ZX Spectrum graphics data
- ZX Spectrum screen formats
- ZX Spectrum Next graphics assets
- Sprite resources
- Tile resources
- Palette resources
- Development-ready binary assets

**Supported File Formats**

- `.png`
- `.scr`
- ZX Spectrum graphics formats
- ZX Spectrum Next graphics formats
- Generated binary resources

**Integration**

- **Assemblers**
  - SjASMPlus
  - PASMO
- **Compilers**
  - z88dk
  - Boriel BASIC
- **Frameworks & SDKs**
  - Compatible with Spectrum and Next frameworks
- **External tools**
  - Remy's Sprite, Tile & Palette Editor

**Primary Purpose**  
A browser-based collection of asset conversion utilities for preparing ZX Spectrum and ZX Spectrum Next development resources.
