## Boriel Basic IDE

**Official URL**
https://github.com/boriel-basic/ZXBasicStudio

**Host Platforms**

- Windows
- Linux
- macOS

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 16K
- ZX Spectrum 48K
- ZX Spectrum 128K
- ZX Spectrum +2
- ZX Spectrum +2A/+2B
- ZX Spectrum +3
- ZX Spectrum Next

**Overview**  
Boriel Basic IDE (ZX Basic Studio) is the official integrated development environment for the Boriel BASIC compiler. It provides a modern, project-oriented workspace designed specifically for Sinclair ZX Spectrum software development, combining source editing, project management, compilation, debugging and emulator integration into a single application. The IDE significantly streamlines the development workflow compared with using the compiler from the command line, making it suitable for both newcomers and experienced developers.

The IDE is designed around the Boriel BASIC language while exposing many of the underlying compiler capabilities, including project organisation, build configuration and source-level debugging. It integrates directly with the Boriel BASIC toolchain and is intended to be the primary development environment for creating modern ZX Spectrum and ZX Spectrum Next applications.

**Development Features**

- Project-based development
- Syntax highlighting and code editing
- Integrated build system
- Boriel BASIC compiler integration
- Source-level debugging
- Breakpoint management
- Emulator integration
- Project explorer
- Build configuration management
- Error and diagnostic reporting

**Supported File Formats**

- `.bas`
- `.bi`
- `.asm`
- `.tap`
- `.tzx`
- `.sna`

**Integration**

- **Assemblers**
  - Boriel BASIC inline Z80 assembler
- **Compilers**
  - Boriel BASIC
- **Emulators**
  - Integrated debugging workflow with supported ZX Spectrum emulators
- **Frameworks & SDKs**
  - Boriel BASIC standard libraries
- **External tools**
  - Configurable external build tools and emulators

**Primary Purpose**  
A fully integrated IDE for developing, building and debugging Boriel BASIC applications for the Sinclair ZX Spectrum and ZX Spectrum Next.

## Klive IDE

**Official URL**
https://dotneteer.github.io/kliveide/

**Source Code**
https://github.com/Dotneteer/kliveide

**Host Platforms**

- Windows

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 48K
- ZX Spectrum 128K
- ZX Spectrum +2
- ZX Spectrum +2A/+2B
- ZX Spectrum +3
- ZX Spectrum Next

**Overview**  
Klive IDE is a modern integrated development environment centred around assembly language development for the Sinclair ZX Spectrum family. It combines a powerful editor with an integrated emulator, assembler support, debugging facilities and project management, allowing developers to edit, build, execute and debug software without leaving the IDE.

Unlike traditional workflows that require multiple standalone applications, Klive IDE provides a unified environment where source code, emulator state, debugging information and build output are available simultaneously. The IDE is particularly well suited to ZX Spectrum Next development while continuing to support classic Sinclair ZX Spectrum machines, making it valuable for both retro and modern homebrew development.

**Development Features**

- Project management
- Integrated source editor
- Syntax highlighting
- Assembly language development
- Integrated emulator
- Source-level debugging
- Breakpoints
- Memory inspection
- Register inspection
- Disassembler
- Build automation
- Configurable toolchain integration

**Supported File Formats**

- `.asm`
- `.tap`
- `.tzx`
- `.sna`
- `.z80`
- `.nex`

**Integration**

- **Assemblers**
  - SjASMPlus
- **Compilers**
  - External compiler support
- **Emulators**
  - Integrated Klive emulator
- **Frameworks & SDKs**
  - Compatible with external ZX Spectrum libraries and frameworks
- **External tools**
  - Configurable external build utilities

**Primary Purpose**  
A modern integrated development environment for Z80 assembly development, debugging and testing across the Sinclair ZX Spectrum family, including the ZX Spectrum Next.
