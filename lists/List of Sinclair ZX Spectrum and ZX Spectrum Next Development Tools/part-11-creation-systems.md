## Kwyll

**Official URL**
https://kwyll.com/

**Host Platforms**

- Windows
- Linux
- macOS
- Web

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 48K
- ZX Spectrum 128K

**Overview**  
Kwyll is a modern interactive fiction authoring system designed for creating text adventures and narrative-driven games for retro computing platforms, including the Sinclair ZX Spectrum. It provides a high-level content creation environment that allows authors to focus on story design, locations, objects, puzzles and game logic without requiring detailed knowledge of low-level programming techniques.

The system generates complete interactive fiction games from structured source content and includes tooling intended to simplify the authoring process. Kwyll is particularly valuable for developers interested in adventure game creation, educational software and narrative experiences where content creation is more important than direct hardware programming.

## Multi-Platform Arcade Game Designer (MPAGD)

**Official URL**
https://jonathan-cauldwell.itch.io/multi-platform-arcade-game-designer

**Host Platforms**

- Windows

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 48K
- ZX Spectrum 128K

**Overview**  
Multi-Platform Arcade Game Designer (MPAGD) is a complete game-authoring environment created by Jonathan Cauldwell for building arcade-style games across several retro computing platforms, including the Sinclair ZX Spectrum. It provides an integrated workflow that combines level design, graphics creation, event scripting and game logic definition into a single development system.

MPAGD is intended to enable game creation without requiring extensive assembly language or compiler knowledge. Developers define game behaviour through the system's scripting and design tools while the engine handles the underlying runtime implementation. This makes it particularly attractive to hobbyists, educators and rapid-prototyping workflows.

- Platform-specific runtime layer

**Development Features**

It provides tools for constructing stories, defining locations, managing inventory systems and implementing game logic through a structured authoring workflow.

The system is designed to lower the barrier to entry for adventure game development by providing specialised tools focused on storytelling rather than low-level programming. Authors can concentrate on content creation while StoryCraft generates the runtime structures required by the target platform.
