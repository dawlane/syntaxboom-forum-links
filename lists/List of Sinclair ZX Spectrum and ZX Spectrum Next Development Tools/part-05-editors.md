## GrafX2

**Official URL**
http://grafx2.chez.com/

**Source Code**
https://github.com/pulkomandy/GrafX2

**Host Platforms**

- Windows
- Linux
- macOS

**Verified ZX Spectrum or ZX Spectrum Next Targets**

- ZX Spectrum 16K
- ZX Spectrum 48K
- ZX Spectrum 128K
- ZX Spectrum +2
- ZX Spectrum +2A/+2B
- ZX Spectrum +3
- ZX Spectrum Next

**Overview**  
GrafX2 is a mature bitmap graphics editor designed specifically for pixel art creation. Although not exclusive to Sinclair systems, it is widely used by ZX Spectrum and ZX Spectrum Next developers for producing sprites, tiles, character sets, user interface graphics and full-screen artwork. Its extensive palette controls, animation support and pixel-perfect editing tools make it one of the most commonly used graphics packages in retro game development.

For ZX Spectrum projects, GrafX2 is frequently used as the primary artwork creation environment before assets are converted into Spectrum-specific formats using dedicated conversion utilities. ZX Spectrum Next developers particularly benefit from its advanced palette handling and support for larger colour spaces.

**Development Features**

- Pixel-perfect drawing tools
- Indexed colour editing
- Palette management
- Tile editing
- Sprite editing
- Animation preview
- Layer support
- Brush management
- Zoom editing
- Script support

**Supported File Formats**

- `.png`
- `.bmp`
- `.gif`
- `.pcx`
- `.iff`
- Native GrafX2 formats

**Integration**

- **Assemblers**
  - Asset export workflows
- **Compilers**
  - Asset export workflows
- **Frameworks & SDKs**
  - Compatible with Spectrum graphics frameworks
- **External tools**
  - Gfx2Next
  - Remy's ZX Spectrum Tools

**Primary Purpose**  
A professional pixel-art editor used to create graphics assets for ZX Spectrum and ZX Spectrum Next software projects.

**Official URL**

- ZX Spectrum 48K

- ZX Spectrum +2A/+2B
- ZX Spectrum +3

**Overview**  
ZX-Edit is a browser-based graphics editor focused on the attribute-based graphics system of the classic Sinclair ZX Spectrum. It allows artists and developers to create screen graphics while working within the colour and attribute constraints imposed by original Spectrum hardware.

The editor provides a convenient web-based workflow that requires no installation, making it useful for quick asset editing, experimentation and educational purposes. It helps developers visualise how graphics will appear on real hardware while maintaining compatibility with classic machine limitations.
